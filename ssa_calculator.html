How It's Calculated (Per Year)

Growth = Start Balance × Investment Return
Expenses = Previous Year's Expenses × (1 + Inflation Rate)
SS Income (COLA) = Previous Year's SS Income × (1 + Inflation Rate)
Net Change = Growth + SS Income - Expenses
End Balance = Start Balance + Net Change
